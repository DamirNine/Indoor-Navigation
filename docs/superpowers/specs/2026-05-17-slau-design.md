# СЛАУ — Решатель систем линейных алгебраических уравнений

**Дата:** 2026-05-17  
**Статус:** Approved

---

## Обзор

Веб-приложение для ввода, составления и решения систем линейных алгебраических уравнений (СЛАУ). Ориентировано на задачи ТММ (Теория машин и механизмов). Работает на ПК и мобильных устройствах, деплоится на GitHub Pages.

---

## Ввод уравнений

### Свободный текстовый ввод
- Пользователь пишет уравнение произвольным текстом: `3x + 2y = 7`
- Любая последовательность букв (латиница + греческий) автоматически распознаётся как переменная
- Переменные подсвечиваются синим цветом и становятся кликабельными прямо в строке ввода
- Числа и операторы (`+`, `-`, `=`) остаются как обычный текст

### Индексы переменных
- Клик по переменной → всплывающее меню с двумя полями: **нижний индекс** и **верхний индекс** (оба опциональны)
- Индексы отображаются у переменной сразу в строке ввода: `x₁₂²`
- Закрытие меню — клик за пределами или Escape

### Управление строками
- Изначально одна пустая строка
- Кнопка **«+ уравнение»** добавляет строку снизу
- Кнопка **«✕»** у каждой строки удаляет её с подтверждением (`window.confirm`)
- Кнопка **«↩»** (Undo) и **«↪»** (Redo) — видимые кнопки в тулбаре, также Ctrl+Z / Ctrl+Y на ПК
- Кнопка **«Стереть всё»** — очищает все уравнения с подтверждением, возвращает к одной пустой строке

## Панель символов ТММ

- Кнопка **«Ω Символы ТММ»** в тулбаре, по умолчанию скрыта
- Открывает боковую панель (справа на ПК, снизу на мобильном) с символами, сгруппированными по назначению:
  - **Угловые:** ω, φ, α, β, θ, ψ
  - **Скорости/ускорения:** ε, δ, λ, ν
  - **Геометрия:** ρ, μ, σ, τ, Δ, Φ
  - **Прочие:** η, γ, ζ, ξ, χ, κ, π, Σ, Π, Γ, Λ, Ξ, Ω
- Клик по символу вставляет его в активное уравнение как переменную (курсор ставится после вставленного символа)

## Специальные правила СЛАУ

### Правило x₁₂ / x₂₁
Если две переменные имеют одинаковое имя и одинаковые цифры в нижнем индексе, но в **обратном порядке** (например `x₁₂` и `x₂₁`):
- Считаются одной переменной: `x₁₂`
- При составлении матрицы коэффициент при `x₂₁` умножается на **-1**
- В ответе показывается: `x₂₁ = −x₁₂ = −2`

### Обработка ошибок
- **Недоопределённая система** (уравнений меньше чем переменных): предупреждение с пояснением
- **Несовместная система** (нет решений): предупреждение, показывается rank(A) и rank(A|b)
- **Слишком много переменных** (> 20): предупреждение до запуска решения

## Решение

### Алгоритм
- Метод Гаусса (прямой + обратный ход) с частичным выбором ведущего элемента
- Работа с дробями через точную арифметику (библиотека `fraction.js`)

### Кнопка «Решить»
- Составляет расширенную матрицу из введённых уравнений
- После решения уравнения остаются редактируемыми — можно изменить и нажать «Решить» снова

### Вывод результата

**По умолчанию — только ответ:**
```
Ответ:
x₁₂ = 2
x₂₁ = −2  (= −x₁₂)
x₃  = 1
```

**Кнопка «Показать решение» раскрывает:**
1. Расширенная матрица системы (таблица с заголовками-переменными, вертикальная черта перед столбцом b)
2. Шаги метода Гаусса: операция над строкой (`R2 ← R2 − (4/3)·R1`) + результирующая матрица
3. Обратный ход: подстановка значений
4. Ответ (всегда внизу, даже при открытом решении)

Кнопка **«Копировать ответ»** — копирует ответ в буфер обмена в текстовом формате.

## Сохранение состояния

- Все уравнения (текст + индексы) сохраняются в **localStorage** автоматически при каждом изменении
- При обновлении страницы уравнения восстанавливаются
- Каждое устройство/браузер хранит своё независимое состояние

---

## История решений

- Последние **10 решённых систем** сохраняются в localStorage
- Кнопка **«История»** в тулбаре открывает список: дата/время + первое уравнение системы как превью
- Клик на запись из истории загружает ту систему в редактор (с подтверждением если текущие уравнения не пусты)

---

## Undo / Redo

- Хранит историю изменений в памяти (не в localStorage)
- Отменяет: добавление/удаление строк, изменение текста уравнений, изменение индексов
- Кнопки **↩ ↪** видны в тулбаре на всех устройствах
- Горячие клавиши Ctrl+Z / Ctrl+Y на ПК
- Глубина истории: 50 шагов

---

## Интерфейс и адаптивность

- **ПК:** боковая панель символов справа, широкая таблица матрицы
- **Мобильный:** панель символов снизу (drawer), горизонтальный скролл таблицы матрицы
- **Zoom:** pinch-to-zoom на мобильном (стандартный viewport), scroll zoom на ПК через CSS transform на области результата

---

## Технологии

- **React + Vite + TypeScript**
- **fraction.js** — точная арифметика дробей
- **localStorage** — персистентность уравнений и истории
- **GitHub Pages** — деплой через `gh-pages` ветка, новый репозиторий `SLAU`
- Никаких бэкендов, полностью статическое приложение

## Что НЕ входит в объём
