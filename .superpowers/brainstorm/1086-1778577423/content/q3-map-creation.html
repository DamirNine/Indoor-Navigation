Как создаются и обновляются карты зданий?

От этого зависит, нужен ли отдельный инструмент для администратора

A

Разработчик готовит карту вручную (JSON/файл)

Схема помещения описывается в структурированном файле — узлы (кабинеты, лестницы, лифты) и рёбра (коридоры). Файл упаковывается в приложение или загружается при первом запуске. Просто, но требует технических знаний.

B

Визуальный редактор карт (web-инструмент)

Администратор рисует план этажа, расставляет точки, соединяет их — экспортирует файл для приложения. Удобно для нетехнических пользователей.

C

Импорт из SVG/AutoCAD/изображения + разметка точек

Загружается готовый план этажа (архитектурный чертёж или фото), поверх него расставляются точки навигации. Реалистичная карта без рисования с нуля.
