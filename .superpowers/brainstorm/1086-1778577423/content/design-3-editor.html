Рабочий процесс администратора

  1. Создать здание, задать название и ID
  2. Добавить этажи, загрузить изображение плана каждого этажа
  3. Расставить узлы: кликнуть на план → выбрать тип (кабинет / лестница / лифт / вход) → ввести название
  4. Провести рёбра: выбрать инструмент «Ребро» → кликнуть два узла → выбрать тип соединения
  5. Для лестниц и лифтов: связать узел на этаже N с узлом на этаже N+1 через диалог «Межэтажное соединение»
  6. Нажать «Экспорт» → скачать building.json + папку с изображениями (или building.zip)

Стек React редактора

Ключевые библиотеки
  • React + TypeScript
  • Konva.js — canvas для карты, drag-and-drop узлов
  • Zustand — управление состоянием
  • Vite — сборка
Ключевые фичи
  • Drag-and-drop узлов по плану
  • Zoom/pan карты
  • Предпросмотр маршрута перед экспортом
  • Валидация графа (нет изолированных узлов)
