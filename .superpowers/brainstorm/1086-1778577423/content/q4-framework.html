Какой мобильный фреймворк использовать?

Оба варианта дают Android + iOS из одного кода и работают полностью офлайн

A

React Native

JavaScript/TypeScript. Огромная экосистема, быстрый старт, много готовых компонентов. Рендеринг карт — через react-native-svg или react-native-maps (офлайн). Хорошо подходит если есть JS-опыт.

B

Flutter (рекомендую)

Dart. Отличная производительность, встроенный Canvas для рисования карт, единый UI на всех платформах. flutter_map + офлайн-тайлы работают «из коробки». Лучший выбор для кастомного рендеринга.
